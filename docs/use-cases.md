# Use Cases

## Use Case 1: View Available Rooms
1. User selects a date and time
2. System shows available rooms

## Use Case 2: Book a Room
1. User logs in
2. Selects a time and room
3. Clicks "Book"
4. Gets confirmation

## Use Case 3: Cancel a Booking
1. User goes to “My Bookings”
2. Selects a booking
3. Clicks "Cancel"
